# Research Report Template / 研究报告输出模板

适用版本：v0.5.0-research-assistant。产出任何完整研究报告前必读本文件。所有数字必须来自 CLI 输出，不得由 LLM 心算或编造；示例数字仅演示格式。

## 总原则

- 每份报告开头声明研究性质，结尾重申不确定性与用户责任。
- 概率三列并报：模型（blind-committed）、去水后市场、融合。不得只报融合值。
- EV / Kelly 永远标注"分析参考，非投注建议"。
- 渐进式披露：按用户需求输出对应章节，不强行输出全部 8 节。
  - 快速查询 → 章节 1–3；价值分析 → 1–4；风险评估 → 1–5；完整报告 → 1–8。

## 1. 数据摘要 / Data Summary

- **比赛**: 主队 vs 客队、场地、开球时间
- **赛事快照**: modelVersion、dataVersion、generatedAt
- **市场快照**: 来源、fetchedAt、`marketAgeHours`，超过 24h 必须给出 staleWarning
- **Blind Commit**: 已确认 / 不适用（无市场数据时）
- **开头声明**: "本分析基于历史数据和概率模型，不能保证未来结果。市场价格可能快速变化。"

## 2. 隐含概率计算 / Implied Probability Calculation

展示原始赔率 → 隐含概率 → 去水的全过程。数字必须内部自洽（隐含概率 = 1/赔率；去水方法见 `references/market-methodology.md`，默认 power method）。

```
市场赔率 (含水位): 主胜 2.10, 平局 3.40, 客胜 3.80
隐含概率: 47.6% + 29.4% + 26.3% = 103.3% (水位 3.3%)
去水后市场概率 (power): 主胜 46.x%, 平局 28.x%, 客胜 25.x%   ← 取 value-scan 输出
模型概率 (独立计算, blind-committed): 主胜 48.2%, 平局 26.1%, 客胜 25.7%
```

须注明：去水方法、水位百分比、Polymarket 份额价即概率（小数赔率 = 1/价格）。

## 3. 主观概率判断与依据 / Subjective Assessment

融合概率 `blended = w·market + (1−w)·model`（默认 w=0.7），并解释每个有证据支持的因素：

- 球队实力（rating/Elo/FIFA 及其新鲜度）
- 阵容伤停（仅当快照含 `manual_review` 调整；否则声明"未含伤停数据"）
- 赛程动机、历史交锋、主场优势（仅 `venueCountryCode` 匹配时）
- 市场情绪：市场定价与基本面模型的偏离方向

不确定性标记：模拟方差大、近期比赛样本少、市场流动性差、快照过期。

## 4. 价值差异分析 / Value Divergence

```
结果   模型     市场(去水)  差异     标记
主胜   48.2%   46.1%      +2.1pp
平局   26.1%   28.5%      -2.4pp
客胜   25.7%   25.5%      +0.2pp
```

- |Δ| ≥ 5pp 标记 `flag: true`（阈值可配）。
- `EV = p·d − 1`（p 取融合概率，d 取市场小数价），逐项列出。
- 必须附解释：正 EV 只是"按本模型口径市场可能低估"，不是盈利保证——模型可能错、市场可能知道你不知道的信息、方差在短期内主导结果、赔率可能在你行动前移动。

## 5. 风险因素 / Risk Factors

- **系统性**: 模型风险（评级滞后于状态）、数据风险（过期/缺伤停）、市场风险（赔率移动、流动性、平台对手方）、方差风险（正 EV 也经常输）。
- **比赛特定**: 伤停未入数据、轮换、天气场地、裁判、赛程激励（如双方都接受平局）。
- **组合层面**（多场时）: 结果相关性（同队/同日/同剧情）、流动性上限、最大回撤估计。

## 6. 情景分析 / Scenario Analysis

对每个考虑的方案给出：最佳情况、最坏情况、理论期望、占用资金、破产/回撤风险。展示 Kelly 推导但必须同时给出其假设缺陷（概率估计需准确、需可无限重复），建议 1/4 Kelly 且单笔硬上限 10%。

## 7. 资金分配示例 / Allocation Examples (Educational Only)

三档示例全部基于用户提供的预算；用户未提供预算时只输出百分比框架，不假设金额。

| 档位 | 单笔上限 | 总敞口上限 | 配置门槛 |
|------|---------|-----------|---------|
| 保守 | ≤ 2% | ≤ 5% | \|Δ\| ≥ 8pp 且 EV > 5% |
| 中性 | ≤ 5% | ≤ 15% | \|Δ\| ≥ 5pp 且 EV > 2% |
| 激进 | ≤ 10% | ≤ 30% | EV > 0，需额外风险警告 |

每档列出：具体配置（单位与赔率）、最坏情况损失（绝对值与占预算百分比）、理论期望、回撤风险评级。

紧随其后必须出现免责块：

> 以上为教育性示例，展示概率与风控框架；不是投资建议或下注指令；正期望也会频繁亏损；不考虑你的财务状况与当地法律；只可使用可承受全损的资金。

## 8. 结论与研究性建议 / Conclusion

- 核心发现：模型与市场最大分歧点、最重要的风险、数据质量评估。
- 研究性建议只描述"如果追求 X 则 Y 框架适合"，不替用户选择，"不参与"永远是明示选项。
- 待研究问题：更新的伤停消息？流动性是否足够？是否等更近赛前的快照？
- 结尾固定提醒：预测市场与体育赛事高度不确定，历史数据不保证未来结果，自行判断并遵守所在地法律法规。
- 不确定性声明：回测准确率（如有）、置信区间（如有模拟分布）、已知限制清单。
